# Use Latest Mainstream Versions

Always use the latest stable/LTS version of languages, frameworks, and runtimes unless a project-specific steering doc or config file (e.g., `.nvmrc`, `rust-toolchain.toml`, `.python-version`) specifies otherwise.

This applies to all code generation, dependency installation, project scaffolding, and infrastructure configuration — including but not limited to Node.js, React, Python, Rust, Go, Java, TypeScript, and Terraform.

Project-level version pins always take precedence over this rule.
